# Out of Scope — MVP — Task A05

These are explicitly **not** built for the hackathon MVP (from the Product Guideline).
Each will be filed as a GitHub issue labeled `out-of-scope` once `gh auth login` is complete.

- [ ] Payroll, attendance, recruitment, appraisal, or promotion features
- [ ] Automatic assignment without human confirmation — the manager always decides
- [ ] Reading private Slack / Teams messages or any conversation content
- [ ] Training a custom ML model
- [ ] Enterprise SSO and production HR-system integrations
- [ ] Predicting protected traits or psychological diagnoses

**Scope rule:** if a feature does not improve the **create → rank → decide → audit** loop, postpone it. A polished allocation loop beats a sprawling HR platform.
