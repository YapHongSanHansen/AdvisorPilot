# Project Atlas

> AI-assisted manpower allocation — assign the right people to projects without overloading the same top performers, while giving leadership visibility into capacity, fit, and repeated override patterns.

**ImagineHack 2026 · Track 3: DoubleDot — Smarter Resource Management.**
Resource = **manpower**. Waste targeted = idle talent, overload / burnout risk, and manual planning time.

## The loop

**Create → Rank → Decide → Audit.** A manager creates a project and its requirements; the system filters out unavailable people and ranks the rest by skills, capacity, work-style fit, and workload distribution; the manager sees a plain-language reason for each recommendation and accepts or overrides; every decision is recorded and a leadership dashboard updates.
